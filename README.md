# Kawaii Life Planner 🎀

かわいいスケジュール帳アプリです！

## Vercelへのデプロイ方法

このフォルダ全体を **Vercel** にデプロイすることで、スマホアプリとしてインストールできるようになります。

### 手順

1.  [Vercel](https://vercel.com/) にログインします（未登録の場合は登録してください）。
2.  ダッシュボードの「Add New...」 -> 「Project」をクリックします。
3.  **このフォルダ（予定表）** をそのまま画面にドラッグ＆ドロップします。
4.  設定はそのままで「Deploy」ボタンを押します。
5.  デプロイが完了するとURLが発行されます。そのURLをスマホで開いてください。

### スマホへのインストール方法

1.  発行されたURLをスマホ（SafariまたはChrome）で開きます。
2.  **iPhoneの場合**: 「共有」ボタン -> 「ホーム画面に追加」を選びます。
3.  **Androidの場合**: メニュー -> 「アプリをインストール」または「ホーム画面に追加」を選びます。

### 注意点

*   **PWA対応**: `sw.js` と `manifest.json` が含まれているので、オフラインでも動くアプリとしてインストールできます。
*   **データのバックアップ**: データはブラウザに保存されます。機種変更などの際は「Settings ⚙️」からバックアップを取ってください。
